# Cream cheese fermentation

This folder contains the following files:
 
* [cream_cheese_fermentation_paper.ipynb](cream_cheese_fermentation_paper.ipynb): 
  This notebook is the starting point of the model. It contains the scientific 
  background of the problem. This is an informal input, personal to the 
  scientist.
* [cream_cheese_fermentation_model.py](cream_cheese_fermentation_model.py): This 
  file contains the model ready to be used with MLFlow.
* [cream_cheese_fermentation.yaml](cream_cheese_fermentation.yaml): This file 
  contains the configuration of the model. It is used to run the model with 
  MLFlow.
* [cream_cheese_fermentation_registry.ipynb](cream_cheese_fermentation_registry.ipynb):
  This notebook is used to register the model in MLFlow. It contains the code 
  to register the model and the model's metadata.

Also there is [plotter.py](plotter.py) which contains an utility to plot the 
results of the model.

> This is a mock project to demonstrate a the business process described in this 
> repository.
